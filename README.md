## 📺 What is this project?
Hand Gesture Controlled YouTube Player lets you control YouTube videos using your webcam and simple hand gestures. No mouse or keyboard needed—just your hand! Built with Python, OpenCV, MediaPipe, and pyautogui.

---

## 🚀 How to Fork & Run
1. **Fork this repo** to your own GitHub account (top right, click Fork)
2. **Clone** your fork: `git clone https://github.com/your-username/this-repo.git`
3. Open the folder in VS Code or your favorite editor
4. Make sure you have Python 3.7+
5. In the project folder, run: `python main.py`
   - The first run will auto-install all required packages
6. Open YouTube in your browser and focus the video
7. Use the gestures below in front of your webcam!

## ✋ Hand Gestures & Actions

| Gesture                | Action         | YouTube Shortcut |
|------------------------|---------------|------------------|
| Fist                   | Pause         | Space            |
| Open hand (all up)     | Play/Resume   | Space            |
| Two fingers up         | Fast Forward  | Right Arrow      |
| Two fingers down       | Rewind        | Left Arrow       |

**How to do the gestures:**
- **Fist:** All fingers folded
- **Open hand:** All fingers up
- **Two fingers up:** Only index and middle up, rest folded
- **Two fingers down:** Only index and middle down, rest up

- On-screen feedback: gesture, action, FPS
- Gesture cooldown to prevent repeated triggers
